# Calculator

A calculator application built with HTML, CSS, and JavaScript.

This project was created to practice JavaScript fundamentals including DOM manipulation, application state management, event handling, and calculator logic.

---


## Features

- Basic arithmetic operations
    - Addition
    - Subtraction
    - Multiplication
    - Division
- Decimal support
- Sign change (+/-)
- Clear (AC)
- Delete (⌫)
- Chained calculations
- Prevents invalid decimal input
- Handles edge cases such as division by zero

---

## Built With

- HTML5
- CSS3
- JavaScript (Vanilla)

---

While building this project I learned:

- DOM manipulation
- Event listeners
- JavaScript state management
- Debugging complex application logic
